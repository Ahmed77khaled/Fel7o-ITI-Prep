# Logo Design Workflow

This skill provides a structured process for creating professional logos that align with a brand's identity and technical constraints.

## Workflow Steps

1. **Brand Identity Analysis**:
   - Extract primary and secondary colors from existing web assets (CSS, manifest.json).
   - Identify core themes, target audience, and key symbols (e.g., educational icons, tech motifs).
   - Review existing visual assets in the repository to ensure consistency.

2. **Concept Development**:
   - Brainstorm 3-5 distinct concepts that represent the brand's mission.
   - For educational/tech projects, consider symbols like targets (🎯), books (📖), graduation caps (🎓), or abstract geometric shapes.

3. **Prompt Engineering for Logos**:
   - Use specific keywords: "minimalist", "flat design", "vector style", "professional", "high contrast".
   - Explicitly mention the color hex codes found during analysis.
   - Specify the background (e.g., "white background" for easy transparency removal).

4. **Generation & Iteration**:
   - Generate multiple variations (at least 3) to provide options.
   - If the user requests refinements, use the best-performing image as a reference.

## Best Practices

- **Aspect Ratio**: Always use `1:1` for logos and app icons.
- **Clarity**: Avoid overly complex details that might get lost at small sizes.
- **Typography**: If text is included, ensure it is legible and uses a modern font style.
- **Consistency**: Maintain the brand's established color palette and tone.
