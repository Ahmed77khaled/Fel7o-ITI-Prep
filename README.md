## 📖 English Overview
**Fel7o ITI Prep** is a state-of-the-art interactive study platform designed to prepare students for the **Information Technology Institute (ITI)** intake examinations. It features a curated, intensive **11-day study plan** covering all essential tracks (Software Development, Web Development, IQ, English, and HR preparation). 

### ✨ Key Features
- **Dynamic 11-Day Schedule:** Chronologically organized daily tasks focusing on key ITI exam requirements.
- **Dual Language Support:** Instant translation toggling between Arabic (العربية) and English (default).
- **Interactive Tracking:** Check off completed tasks with progress indicators, day streaks, and visual feedback.
- **Resource Integration:** Downloadable and previewable study resources (PDFs, templates, and exam materials) for Web/SD tracks, IQ, English, and HR.
- **Sleek UX & Aesthetics:** A premium, fully responsive glassmorphism dark/light design system with custom smooth transitions.
- **Local Storage Sync:** Your progress is automatically saved to your browser so you never lose track.

## 📖 نظرة عامة بالعربية
**منصة فيلحو للتأهيل للـ ITI** هي منصة تفاعلية متكاملة مصممة خصيصاً لمساعدة الطلاب في الاستعداد لاختبارات قبول **معهد تكنولوجيا المعلومات (ITI)**. توفر المنصة خطة دراسية مكثفة ومنظمة مدتها **11 يوماً** تغطي جميع المسارات والمهارات المطلوبة (تطوير البرمجيات، تطوير الويب، اختبارات الذكاء IQ، اللغة الإنجليزية، والمقابلات الشخصية HR).

### ✨ أهم المميزات
- **خطة 11 يوم تفاعلية:** تقسيم زمني دقيق ومهام يومية تركز على متطلبات الاختبارات الفعلية.
- **دعم كامل للغتين:** إمكانية التبديل الفوري بين اللغة العربية والإنجليزية بضغطة زر واحدة.
- **متابعة الإنجاز:** نظام تفاعلي لتحديد المهام المكتملة مع حساب نسب الإنجاز الكلية، وعداد الأيام المتتالية (Day Streak).
- **مستودع ملفات متكامل:** وصول مباشر وتحميل للملفات الدراسية وكتب الـ IQ ونماذج الأسئلة للمقابلات الشخصية والاختبارات التقنية.
- **تصميم عصري وجذاب:** واجهة مستخدم متجاوبة (Dark/Light mode) تعتمد على نمط الـ Glassmorphism وتأثيرات بصرية راقية.
- **حفظ تلقائي للمستوى:** يتم حفظ تقدمك تلقائياً في المتصفح حتى لا تفقد إنجازك.

## 🛠️ Tech Stack | التقنيات المستخدمة
- **Frontend:** Vanilla HTML5 & Javascript (ES6)
- **Styling:** Custom CSS3 with dynamic variables and theme switching mechanics
- **Data Model:** JSON-based state representation (`js/data.js` & `js/config.js`)
- **PWA Ready:** Manifest files configured for native desktop/mobile installation.

## 🚀 How to Run Locally | كيفية التشغيل محلياً
Since the application runs entirely on the client side, you don't need any complex local server setups.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ahmed77khaled/Fel7o-ITI-Prep.git
   ```
2. **Open the project:**
   Simply double-click on [index.html](file:///D:/fel7o%20tech/acc%20iti/index.html) to open the application in your preferred web browser.
